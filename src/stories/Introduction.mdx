### Utility Classes

- **theme-background-color**<br/>
  테마 별 배경 색상을 지정합니다.

  ```css
  .theme-background-color {
    @apply bg-background dark:bg-dark-background;
  }
  ```

- **theme-text-color**<br/>
  테마 별 글자 색상을 지정합니다.

  ```css
  .theme-text-color {
    @apply text-black dark:text-white;
  }
  ```

- **transition-default-all**<br/>
  동일한 duration 값을 사용하기 위한 클래스입니다.

  ```css
  .transition-default-all {
    @apply transition-all duration-300 ease-in-out;
  }
  ```

- **bento**<br/>
  bento grid의 아이템에 동일한 border-radius 값을 사용하기 위한 클래스입니다.

  ```css
  .bento {
    @apply rounded-[24px];
  }
  ```

- **p-bento**<br/>
  bento grid의 아이템에 동일한 padding 값을 사용하기 위한 클래스입니다.
  ```css
  .p-bento {
    @apply p-[24px] lg:p-[32px];
  }
  ```
